# trimmp3
A python script to trim mp3 files

### Dependencies
ffmpeg  
```pkg install ffmpeg```

### Usage
To keep the original file name:  
```python trimmp3.py your_file.mp3 [start time] [end time]```  
This will output your_file_trimmed.mp3  

To change the file name:  
```python mp4tomp3.py your_file.mp3 [start time] [end time] new_file_name.mp3```  
This will output new_file_name.mp3
